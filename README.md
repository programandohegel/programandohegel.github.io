# Máquina Hegeliana

Implementación computacional de la *Fenomenología del Espíritu* de Hegel, basada en la edición de W. Roces (FCE).

> La dialéctica no es una metáfora.  
> Es un algoritmo.

Este repositorio contiene el código fuente de la primera máquina hegeliana: un sistema en Python que ejecuta el movimiento de la conciencia a través de los primeros cinco capítulos de la *Fenomenología*.
